Control what content will be seen by your feed readers. Send unique content to your feed readers by using this plugin.

== Description ==

By using this plugin you can send unique content to your feed readers or subscribers. You can create feed posts which will be only visible to your feed readers or by using a shortcode([premium version](http://muneeb.me/wordpress-feed-plugin/)) insert feed only content in a  new or existing WordPress blog post.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
